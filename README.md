  - 게임설명
  - Class Diagram
  - Demo

> 게임설명
- 이동을 하며 확률적으로 아이템을 얻거나, 좀비를 만나면서 좀비를 물리치는 게임이다.
- 일정 거리 수마다 주인공은 성장을 하며, 좀비를 물리칠수록 좀비도 점점 강력해진다.
- 일정수 좀비를 물리칠 경우 "보스 좀비"도 등장한다.
- 낮은 확률로 크리티컬 공격을 합니다.
